Recursos que utilizamos

HTML (49.4%)

Este lenguaje marca la estructura y el contenido de la página. Define elementos esenciales como la barra de navegación, los menús y la sección de visualización de imágenes, lo que permite que la información esté organizada y sea accesible para los usuarios.

PHP (45.4%)

PHP es un lenguaje de programación del lado del servidor que permite crear contenido dinámico e interactuar con bases de datos, facilitando la gestión de datos y la lógica del servidor en nuestra aplicación web.

JavaScript (3.2%)

Este lenguaje se encarga de la interactividad en la página. Permite funciones dinámicas como la búsqueda de imágenes y la gestión de eventos, como mostrar y ocultar menús, lo que enriquece la experiencia del usuario al interactuar con el sitio.

Docker (2.0%)

Docker nos permite crear, implementar y ejecutar aplicaciones en contenedores, facilitando la consistencia entre diferentes entornos de desarrollo y producción, asegurando que nuestra aplicación funcione de manera idéntica en cualquier sistema.
